Application Name
================
Boot1


Application Version
===================
2.3

NCOS Devices Supported
======================
ALL


External Requirements
=====================
Two SIMs should be installed


Application Purpose
===================
This application will perform a speed test on both installed SIMs
and disable the slowest SIM. This means that only the best SIM will
be enabled after the application has been executed. The application
will only run once.


Expected Output
===============
Once executed, only the best performing SIM will be configured.
