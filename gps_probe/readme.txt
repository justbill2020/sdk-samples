Application Name
================
gps_probe


Application Version
===================
1.0


NCOS Devices Supported
======================
ALL


External Requirements
=====================
- SIM for a carrier that supports GPS
- GPS Antenna connected to the device


Application Purpose
===================
This application will probe the device for GPS information. It will
first check that GPS has been enabled. Appropriate logs will be output
to indicate the GPS location or that it is not enabled.


Expected Output
===============
GPS location output in the device logs.
